Cálculo de Média de Atletas

Este projeto em JavaScript realiza o cálculo da média das notas de atletas.

Funcionalidade

Para cada atleta, o programa:
1. Ordena as notas em ordem crescente
2. Remove a menor e a maior nota
3. Calcula a média das três notas restantes
4. Exibe o nome do atleta, suas notas e a média final no console

Conceitos Utilizados

- Arrays e objetos
- `forEach`
- `sort`
- `slice`
- `reduce`
